# 🏥 MedCare – Healthcare Solutions Website

A modern, responsive multi-page healthcare website built with pure HTML, CSS, and JavaScript — no frameworks required.

## 📁 Project Structure

```
medcare/
├── index.html        # Main HTML file (all pages as SPA)
├── css/
│   └── style.css     # All styles and responsive design
├── js/
│   └── app.js        # Navigation, doctors data, form logic
└── README.md
```

## ✨ Features

- **Single Page Application (SPA)** — smooth page transitions without reloading
- **6 Pages:** Home, About, Services, Doctors, Contact, Appointment
- **Responsive Design** — works on desktop, tablet, and mobile
- **Doctor Search & Filter** — live search by name and specialty
- **Appointment Booking Form** — with time slot selection and confirmation
- **Contact Form** — with success feedback
- **No dependencies** — pure HTML, CSS, JavaScript

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/medcare.git
   ```

2. Open `index.html` in your browser — no build step needed.

## 🎨 Tech Stack

- HTML5
- CSS3 (CSS Variables, Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- Google Fonts (Nunito, Playfair Display)
